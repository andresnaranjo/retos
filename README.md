## Supervised Learning
#### Basico
| Dataset | Descripcion | Contribuyente |
| - | - | - |
| [Iris Dataset](https://github.com/colomb-ia/supervised-basico-iris) | Contruye un clasificador de tipos de plantas | [Sebastian Pineda](https://github.com/sebastianpinedaar) |
